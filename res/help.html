Polished Map

Copyright © 2017 Rangi

A map and tileset editor for Pokémon Polished Crystal.

Inspired by crowdmap, but implemented with C++ and FLTK, and focused on graphics instead of script editing.

Source code is available at https://github.com/roukaour/polished-map.

Most functions are available via the menu bar, the toolbar, or shortcut keys.

The sidebar palette uses the mouse:

  • Left-click to select a metatile.
  • Right-click to edit a metatile.
  • Middle-click and drag to scroll.

The map area also uses the mouse:

  • Left-click or drag to place the metatile selected in the sidebar palette.
  • Right-click to select a metatile from the map.
  • Middle-click and drag to scroll.

The metatile editor window works similarly, except without scrolling.
